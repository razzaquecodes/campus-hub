FAQ

Common questions,
answered clearly

How do I install Campus Hub on iPhone?

Open this website in Safari, tap the Share button (box with an arrow), scroll to "Add to Home Screen", and tap Add. Campus Hub will appear on your Home Screen like any native app.

Does it require the App Store?

No. Campus Hub is a Progressive Web App (PWA). You install it directly from your browser in seconds — no App Store or Play Store visit required.

Can I install it on Android?

Yes. Open this site in Chrome on your Android device. Chrome will show an "Add to Home Screen" prompt. You can also download the native APK directly from the Install section above.

Does it work offline?

Core app features and cached data are accessible offline. Live data like results and marks requires an internet connection to fetch fresh data from MAKAUT servers.

Is my data secure?

Yes. Authentication is verified directly against MAKAUT servers. Credentials are stored locally with encryption via expo-secure-store. Your profile is synced securely via Supabase with row-level security.

Technology

Built with a production stack

Every technology choice made intentionally — for performance, scalability, and developer experience.

Mobile · Frontend
React Native & Expo

Cross-platform mobile development with Expo SDK 54 and Expo Router for file-based navigation. TypeScript throughout for type safety.

react-native expo SDK 54 expo-router typescript
State · Data
Zustand & React Query

Lightweight global state with Zustand. Server state, caching, and async data management handled by TanStack React Query.

zustand @tanstack/react-query
Backend · Database
Supabase & Node.js

Managed PostgreSQL database with Supabase for profiles, combined with a custom Node.js/Render backend for secure MAKAUT academic data orchestration.

@supabase/supabase-js postgresql node.js
Auth · Identity
MAKAUT Verification

Direct student authentication and data verification via MAKAUT APIs. Persistent, encrypted local sessions via expo-secure-store.

makaut integration expo-secure-store async-storage
