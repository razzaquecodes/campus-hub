Technology

Built with a production stack

Every technology choice made intentionally — for performance, scalability, and developer experience.

Mobile · Frontend
React Native & Expo

Cross-platform mobile development with Expo SDK 54 and Expo Router for file-based navigation. TypeScript throughout for type safety.

react-native expo SDK 54 expo-router typescript
State · Data
Zustand & React Query

Lightweight global state with Zustand. Server state, caching, and async data management handled by TanStack React Query.

zustand @tanstack/react-query
Backend · Database
Supabase & PostgreSQL

Managed PostgreSQL database with Supabase. Real-time capabilities, row-level security, and built-in authentication infrastructure.

@supabase/supabase-js postgresql rls policies
Auth · Identity
Google OAuth & Supabase Auth

OAuth 2.0 sign-in via Google, managed through Supabase Auth. Persistent sessions with expo-secure-store and async storage.

google oauth expo-secure-store async-storage
